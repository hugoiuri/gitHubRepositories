# gitHubViewer

Para executar este projeto basta executar os seguintes comandos:

```sh
    $ npm install
    $ bower install
    $ npm start
```
#### Os pacotes npm utilizados neste projeto são:
  - bower
  - http-server

#### Os pacotes bower utilizados neste projeto são
  - angular
  - angular-route
  - angular-resource
